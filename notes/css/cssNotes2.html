Text-Align

    This method allows to specify the horizontal alignmemt of the text

    Example


    Left

    Aligns the text to the left

    Right

    Alligns the text to the right

    Center

    Makes the text centered

Text-Decoration

Adds an underline, overline, or line-through, to any text

Example


underline

Horizontal line underneath text

overline

horizontal line above text

line-through

strikethrough. a hoizontal line through the text

Font-Weight

    adds bold or light text

    Example


    normal

    standard weight. equivilent to size 400

    bold

    bold weight. equivilent to size 700

    100

    lightest text

    900

    boldest text

    Font-Family

    specifies the font for an element

    Example


    serif

    small line attatched to the end of a stroke in a letter

    sans-serif

    does not have extending features at the end of a stroke
