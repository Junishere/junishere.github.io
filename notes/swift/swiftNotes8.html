Collections


Arrays

This is the most common collection type in swift is Arrays which are like an ordered list of same-typed values, these can help specify what type of value be held in collection.

Example:
var cars: [String] = ["Honda","BMW","Audi"]

var counter: [Int] = ["25","30","35"]

Also, the contains(_:) method can be used by passing in the desired value if the array contains the value, the expression is true otherwise it's false.



let courses = ["math","science","english"]
if courses.contains("math") {
print("math is available")
}
