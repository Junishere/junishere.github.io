Loops


A for in loop executes a set of statements for each item within a range of sequence, or a collection, this is useful for repeating like a set of numbers.


Example:

For randomNumbers in 20...25 {
print("\(randomNumbers) has been generated")
}

console output:

20 has been generated
21 has been generated
22 has been generated
23 has been generated
24 has been generated
25 has been generated

We also don't have to put the value in and can just leave it with a "_:" value.



for _ in 1..5 {
print("What's up")
}

what's up
what's up
what's up


The for-in syntax can be used iterate over each item in an array.



let phones = ["Samsung", "IPhone", "LG"] {
for phone in phones
print("My phone is a \(phones)")
}


Since String is a collection type it can also be used in a for in loop because it has the a similar functionalality to an array.


for value in "XYZ" {
print("the letter values are \(value)")
}
