Assigning A Value

Use the math signs +, -, *, /to preform simple math tasks.

Example: var opponent = 3 * 8.

Example 2: myScore = 100 / 4.

This is the same with decimal numbers instead of using whole numbers.

Numeric Type Conversion

Order of Operations

this allows to use order of operation multipliaction, addition, subtraction, and division.

Exapmle:

var x = 3

var y = 4

var z = 5

x + y * z = 23

(x + y) * z = 35
