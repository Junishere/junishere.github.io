Structures

the word struct means that it is listing out the variables and constants declorations with the type of annotation

this is used to define the properties and showing how to use a name property


example:

struct gamerTag {
var realName: String
}

Access data is stored in it's properties such as the person's realName using the dot syntax


let gamer = gamerTag(realName: "Branden")
print(gamer.realName)

console output

Branden
