Defining Function

It's similar to the let, constant or var, variable, however the func keyword tells the Swiftcomplier it's a function

If this function has a return value, you'll write an arrow (->), this follows by the type of data the function returns such as Int, String, etc

Then you can use the parameters to specify like a list when the function is called

Another way we can use the function is by explaining ten digit numbers of pi, we don't need to put a return value for this


Example 1:
func displayPi () {
print("3.1415926535")
}

Once the funtion has been called it can be declared anywhere just by writing the name


displayPi()
- Console Output:
3.1415926535

Parameters

To specify a function with a parameter, insert a name for the value, a colon


Example 2:
func addition(value: Int) {
let result = value + 9
print("if you add \(value) by 1, you'll get \(result).")
}

In the case above, value is a constant that can be used within the function.


addition(value: 10)
"if you add 9 and 1, you'll get 10."

Then to pass multiple arguments to a function you'll need to seperate each parameter with a comma.

The following is an example of a function that takes in two Int parameters, multiply together and print the result.


func multiply(oddNumber: Int, evenNumber: Int) {
let answer = oddNumber * evenNumber
print("result equals to \(answer)")
}

The function is then called liked this


multiply(oddNumber: 24, evenNumber: 35)
"the answer to this is 840"

Argument Labels

So far, the label for each argument within a function has the same name intentionally as it does externally

In the example the firstName is used both within the function and when the function is called


Example 3:
func introduceMyself(firstName: String) {
print("Hello,\(firstName)")
}

sayHello(firstName:"Branden")

The argument labels to and and make the function read very clearly


func sayHi(to brother: String, and friend: String) {
print("Hey \(brother) and \(friend)")
}

If the function is clearer without an argument label, it can be omit for example take the print function


print("Hello there")

The name of the function already makes it very clear that should be pass in a parameter


To omit the argument label, use _in place of the external name:


func introduceFriend(_ friend: String, _ anotherFriend: String,) {
print("This is \(friend) and \(anotherFriend)")
}

introduceFriend("Cal, "Dennis")

Default Parameter Values


Parameters allow the function to remain felxible, but often there's a value to use most of the time

As part of function definition the default value can be provided for any parameter


func display(videoGame: String, place: Int = 2) {
print("\(videoGame): \(place)")
}

display(videoGame: "PacMan", highScore: 400) //"PacMan: 400"
display(videoGame: "PacMan") //"PacMan: 400"
