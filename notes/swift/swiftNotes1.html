Constants and Varibales

Constant is a value where it doesn't change by using the code word let.

Example: let name = "John"


Variables are values that can be changed sometimes during a lifetime using the code word var.

Example: var age = 29

Common Types

Integers = Int: they represent whole numbers like 4 or 6.

Doubles = Double: they represent numbers with decimals like 3.14 or pi.

Boolean = bool: they represent true or false values.

String = string: they represent text like "Hello world."
