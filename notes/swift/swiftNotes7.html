Class

You define a class using the Class keyword along with a unique name. The properties have to be defined as part of the class

by listing the constant or variable declarations with an appropriate type of annotation.


Example:
class item {
let brand: String

init(brand: String) {
self.brand = brand
}
}

Also, we can add function to the class definition.
