Class

You define a class using the Class keyword along with a unique name. The properties have to be defined as part of the class

by listing the constant or variable declarations with an appropriate type of annotation.


Example:
class item {
let brand: String

init(brand: String) {
self.brand = brand
}
}

Also, we can add function to the class definition.


class item {
let brand: String

init(brand: String)
self.brand = brand
}

func priceTag() {
print("Price of item is 5.99")
}
} let item = Item(brand: "Nike")
print(item.brand)

Subclasses

Subclassing is an act of base of a new class on an existing class, and it inherits properties and methods.


class phone: item {
var hasCase = false
}

The inherited properties and methods with the phone value and can set to be true with the hasCase.



Overide Methods and Properties

This method allows the subclass to provide it's own custom implantenation of a property or a method.


class samsung: phone {
override func makeRingSound() {
print("ring ring ring!") }
}

let phone = Phone()
phone.makeRingSound()

You can also override the properties by providing a getter or a block of code that returns a value, like a computed property.


class boat: vehicle {
var gear = 5
override var information: String {
return super.information + "in gear \(gear)"
}
}
