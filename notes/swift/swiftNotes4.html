Concatenation and Interpolation

The "+" operator can combine not only the numbers, but also words like "Hello" or "Hello World"

Example:
Var introduceToFriend = "Yo"
introduceToFriend = introduceToFriend + ", what's up"

The value of the Constant and Variable into String can be used by putting backslash "\" and parentheses around the constant or variable "()"

Second Example:
let name = "Cal"
let item = "hammer"
print("\(name) found a \(item) in the garage")

let firstNumber = 20
let otherNumber = 30
print("if the first number is \(firstNumber) and the second number is \(otherNumber), then firstNumber + otherNumber equals 50")

String Equality and Comparison

Using the "==" operator checks for identical characters in the same order

Also, we can use the "!=" operator to say they are not the same

Example:
let tool = "hammer"
let otherTool = "hammer"

if tool == otherTool
print("They are the same")

if tool != otherTool
print("They are not the same")


Second Example:
let lowercaseLetters = "john cena"
let capitalLetters = "John Cena"

if lowercaseLetters != capitalLetters {
print("They are not the same") }

Lowercased letters

This example can use the ".lowercased()" method to compare the two constants or variables

Third Example:
let item = "Bommerang"
if item.lowercased() == "boomerang".lowercased() {
print("The two names are equal") }

Instead of the "==" operator we can use the "hasPrefix(_:)" or the "hasSuffix(_:)" to match the beginning or ending of the String


Fourth Example:
let introduction = "What's, up"
print(introduction.hasPrefix("What's"))
print(introduction.hasSuffix(", up"))

Another example we can also use is the ".contains(_:)" method


Fith Example:
let cartoonIntro = "Hi there, I'm Bob the builder"

if cartoonIntro.contains("Bob the builder") {
print("Making an introduction to tv show") }

Escape Characters

Double quote: \"

Single quote: \'

Backslash: //
